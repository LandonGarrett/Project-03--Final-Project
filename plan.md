# Project Plan  
## Sustainable Homestead Resource Website

## TAME Risk Analysis

### T — Technical  
- **Risk:** GitHub Pages publishing errors  
- **Mitigation:** Test early; verify repo structure; keep filenames simple

### A — Assumptions  
- **Risk:** Assuming time availability remains stable  
- **Mitigation:** Build buffer time; complete core tasks early

### M — Market / Stakeholders  
- **Risk:** Content may not meet instructor expectations  
- **Mitigation:** Align deliverables with rubric; request feedback early

### E — Execution  
- **Risk:** Scope creep  
- **Mitigation:** Follow scope statement strictly; use change requests if needed
